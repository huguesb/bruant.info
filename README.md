bruant.info
===========

This website is built using [Jekyll](http://jekyllrb.com/).


Source structure
----------------

    ├── pages       special pages
    ├── resources   static resources
    └── _layouts    HTML templates used by Jekyll


Development flow
----------------

1. Edit with text editor

2. Test locally

    jekyll serve

3. Push to github

    git push

4. Deploy

    git push deploy master
